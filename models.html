Cloud models — when you need more

For tasks that need more than any on-device model can deliver today, FRIDAY lets you connect your own API key to access the world's most capable cloud models. Your key goes directly to the provider — FRIDAY is just the interface.

OpenAI

Connect your OpenAI API key to access GPT-4o — one of the most capable language models available. Ideal for complex reasoning, long documents, and advanced coding tasks.

GPT-4o

Anthropic

Use your Anthropic API key to access Claude Sonnet — excellent at nuanced writing, analysis, and thoughtful multi-step reasoning with a very long context window.

claude-sonnet-4-6

Cloud features are fully optional. The default on-device experience never requires an API key or internet connection.
